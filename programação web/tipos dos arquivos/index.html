O que é um arquivo?
Arquivos são espaços alocados em uma unidade de armazenamento que guardam bytes e bytes de conteúdo, textos, imagens, videos e outros documentos.
Extensões de arquivos
TXT(Arquivos de texto) Arquivos .txt são arquivos de texto simples.
EXE(Arquivos executáveis) Arquivos .exe são arquivos que contém um binário (linguagem de máquina) que é possível executar como um programa no computador.
DLL(Arquivo de biblioteca dinâmica) Arquivos .dll ou Dynamic Link Library são arquivos semelhantes aos executáveis, mas a diferença é que é não é possível executar ele como um programa, esses arquivos podem servir de códigos pré-compilados para um programa executável utilizar de suas funções e recursos.
ZIP(Arquivos comprimidos) Arquivos .zip são arquivos e pastas comprimidos em um arquivo "zipado". O arquivo utiliza de criptografia e concatenação de bits para juntar tudo em um arquivo.
PDF(Documento portátil) Arquivos .pdf são documentos que não necessitam de qualquer outra ferramenta para abrir e ler, todos os computadores e celulares tem o Visualizador de PDF por padrão. Os arquivos podem conter textos, imagens e outros elementos visuais.
DOC/DOCX(Arquivos Word) Arquivos .doc (documento Word)/.docx (XML do Documento Word) são documentos feitos no Microsoft Word ou em qualquer programa que tenha suporte ao Office Open XML. O .docx é compativel apenas com versões modernas do Word, enquanto o .doc é compativel com versões mais antigas do Word
PPT/PPS(Apresentação PowerPoint) Arquivos .ptt/.pps são arquivos de apresentação do Microsoft PowerPoint. O arquivo .ppt contém textos, imagens e outros elementos visuais e é editável, diferente do .pps que serve apenas para apresentar, o arquivo configura para automaticamente apresentar o .ppt.
SLX/XLSC(Arquivos Excel) Arquivos .slx/.xlsc são arquivos do Microsoft Excel. Os arquivos .xlsc tem uma planilha enorme onde os dados são em formato XML, equanto o .slx representa gráficos.
JPG/JPEG(Arquivos de imagem) Arquivos .jpg/.jpeg são arquivos de imagem, sendo sua maioria fotografias leves e com cores sólidas, então esse formato de imagem não oferece suporte ao fundo transparente.
GIF/PNG(Arquivos de imagem) Arquivos .gif/.png são arquivos de imagem. Os arquivos .gif e .png oferecem suporte ao fundo transparente, mas .gif são sequências de imagens, gerando uma animação, enquanto o .png é uma única imagem estática.
HTML/HTM(Arquivos de hipertexto) Arquivos .html/.htm são arquivos de marcação de hípertexto, os arquivos são interpretados pelos navegadores por padrão, é responsável por estruturar um website.
MP3/WAV/MID(Arquivos de audio) Arquivos .mp3/.wav/.mid são arquivos de áudios com diferentes níveis de supressão do peso do áudio.
AVI/MPG/WMV/MOV(Vídeo) Arquivos .avi/.mpg/.wmv/.mov são arquivos de vídeo com diferentes níveis de qualidade e supressão de imagem e com diferentes métodos de interpretação de imagem e áudio.
CSS(Cascading Style Sheets) Arquivos .css são arquivos de estilização para arquivos .html (marcação de hípertexto).
JS(JavaScript) Arquivos .js são arquivos que contém códigos da lionguagem de programação "JavaScript".
SVG(Arquivos de imagem) Arquivos .svg são imagens "desenhadas" e simples e modeláveis com edição de imagens utilizando HTML e CSS.
PHP(Arquivos PHP) Arquivos .php são arquivos que contém códigos da linguagem de programação "PHP" para páginas dinâmicas.
ASPX(Arquivos ASP.NET) Arquivos .ASPX são páginas web que são geradas pelo ASP.NET CORE (com Razor e C#).
SQL(Arquivos PHP) Arquivos .sql são arquivos que contém queries do "SQL" para ser uma forma rápida de executar uma query padrãos no banco de dados.
PSA(Arquivos do Photoshop) Arquivos .psa são arquivos que contém conteúdo interpretável pelo Adobe PhotoShop para edição do arquivo principal, com arquivos de origem da edição.
CDR/AI(Arquivos CorelDraw) Arquivos .cdr são arquivos que contém conteúdo interpretável pelo CorelDraw para edição do arquivo principal, com arquivos de origem da edição. Os arquivos .ai são arquivos de ilustração do Adobe Illustrator.
XML(Arquivos XML) Arquivos .xml são arquivos que contém notação de objetos para linguagens e programas interpretarem.
JSP/CFM(Arquivos JavaServer Page) Arquivos .jsp são arquivos de JavaServer Page, que utiliza do Java para criar páginas dinâmicas, semelhante ao PHP.
TIF(Tagged Images File) Arquivos .tif são gráficos raster e informações de imagem.
FLA/SWF(Arquivos Flash Animation) Arquivos .fla e .swf são arquivo de animação do Flash.
DWG(Arquivos AutoCad) Arquivos .fla são arquivo que contém gráficos do AutoCad.
